# NanoFlow Studio
面向数据产品经理的低代码工作流编排工具，可快速搭建数据采集、清洗、可视化的自动化流程，帮你摆脱重复的手工数据操作。

## 核心功能
- 拖拽式流程编排，无需复杂代码即可搭建数据链路
- 支持主流数据库、API、本地文件等多源数据接入
- 内置数据清洗、转换、可视化组件
- 自定义脚本节点，满足个性化数据处理需求

## 快速开始
1. 下载Releases中的绿色版安装包，解压即可运行
2. 复制`config.example.ini`为`config.ini`，填入你的服务器和密钥信息
3. 拖拽组件搭建第一个数据工作流

## 已知问题（不影响基础使用）
1. 上传大文件时偶尔会出现超时提示，刷新后可继续操作
2. 批量处理数据时进度条显示不准确，但实际任务会正常完成
3. 更多问题请查看Issues
